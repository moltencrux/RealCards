## Configuration

### Section: card
This is for card-related settings.

- `fontSize` — **Base font size in px** of the card: `12`.

- `minFontSize` — **Minimum font size in px** for cards with content overflows: `12`.

- `padding` — Padding of the card: `0.8em`. Can be set to `0` to have no space between cards.

- `reverseOrderOnBack` — Whether the question/answer order is reversed on the back (useful for printing): `true` / `false`.

- `showBorders` — Show dotted borders between cards (helps manual cutting): `true` / `false`.

- `skipQuestionOnBack` — Do not repeat the question on the answer card: `true` / `false`.

### Section: paper
This is for paper and printing settings.

- `cardsPerCol` — Number of cards displayed vertically: `6`.

- `cardsPerRow` — Number of cards displayed horizontally: `3`.

- `format` — Paper format for your region: `A4` / `letter`.

- `orientation` — Paper orientation: `landscape` / `portrait`.
